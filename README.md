# VenueHub - Backend

Backend de l'application VenueHub, développé avec ASP.NET Core et Entity Framework.

## Fonctionnalités
- Authentification JWT
- Gestion des utilisateurs et événements
- Réservation de salles
- API REST complète

## Stack
- ASP.NET Core 8
- Entity Framework Core
- SQL Server
- Swagger
